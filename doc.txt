dwspgen - Random Password Generator

Описание:
Гененирует пароль с определенной длинной начиная с 8 символов
и до 40 (пользователь сам выбирает длину). Нужные элементы
в пароле следует выбрать с помощью чекбоксов. Должна присутствовать
функциональность буфера обмена, то есть класть пароль в буфер
при нажатии на кнопку и чистить буфер от пароля при нажатии.

Символы:
По дефолту в пароле присутствуют символы большого, маленького
регистра. Можно включить цифры и специальные символы с помощью
чекбоксов

Из элементов:
Окно без ресайза

Пустое поле в котором будет появляться пароль
после нажатия на кнопку генерации

Две кнопки, которые касаются буфера и главная кнопка генерация
Два чекбокса, включение цифр и специальных символов
И регулировка длины пароля

Сделать всё небольших размеров без объемных отступов
Добавить хоткей CTRL + Q - выход из приложения
